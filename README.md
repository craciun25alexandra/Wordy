# Wordy
PCLP3 project
Este un mini-game in terminal in care este ales random un cuvant de ghicit de 5 litere din 3 incercari,
ideea fiind asemanatoare cu jocul Wordle. Se da ca input un cuvant, iar daca litera exista dar nu pe acea
pozitie va arata Y(yellow) pe pozitie, G(green) daca este pozitia corecta, vor fi pastrate si afisate literele
folosite pt a nu le reutiliza, iar daca inputul nu exista in lista sau nu respecta conventiile(de 5 litere),
nu se va lua din numarul de incercari.
